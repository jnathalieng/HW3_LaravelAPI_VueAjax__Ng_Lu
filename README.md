# Cartoon Network App

This project is a Vue front-end application connected to a Laravel API.
The layout and structure follows the Bookstore class build example.
The application displays Cartoon Network shows and characters using data from the API. Each show includes a cover photo and a list of characters.  
Each character includes an image and description. The project uses Vue for dynamic content rendering, GSAP for simple animation, and SASS for styling.


## Technologies used:

Vue  
GSAP  
SASS  
Fetch API  
CSS Grid (grid.css)  
Laravel API  
MySQL  


## Installation

Requires:
Node.js  
PHP  
Composer  
MySQL  

Run the Laravel API using:
php artisan serve

Run SASS using:
sass -- watch sass:css

Open the front-end using a local server.
